# 📒 สมุดบัญชีภาษี — Tax Ledger

**เครื่องคำนวณภาษีไทยครบวงจร ทำงานได้ 100% ในเบราว์เซอร์ ไม่มีเซิร์ฟเวอร์ ไม่มีการส่งข้อมูลออกจากเครื่องของคุณ**

เปิดไฟล์เดียว ใช้งานได้ทันที — คำนวณภาษีเงินได้บุคคลธรรมดา ภาษีมูลค่าเพิ่ม และภาษีนิติบุคคล พร้อมปฏิทินแจ้งเตือนกำหนดยื่นภาษี และระบบเก็บประวัติการคำนวณ ครบในที่เดียว

---

## ✨ ทำไมต้องใช้ตัวนี้

- **🧮 คำนวณภาษีครบ 3 แบบในที่เดียว** — บุคคลธรรมดา (PIT), มูลค่าเพิ่ม (VAT), และนิติบุคคล พร้อมอัตราภาษีปีล่าสุด ไม่ต้องสลับไปมาระหว่างหลายเว็บ/หลายไฟล์
- **🔒 ข้อมูลไม่มีวันหลุด** — ไม่มี backend ไม่มีการส่งเลขรายได้หรือข้อมูลส่วนตัวออกจากเครื่องแม้แต่ไบต์เดียว ทุกอย่างเก็บในเบราว์เซอร์ของคุณเท่านั้น เหมาะกับข้อมูลการเงินที่ละเอียดอ่อน
- **⚡ ไม่ต้องติดตั้งอะไรเลย** — ไม่มี framework, ไม่มี build step, ไม่ต้องมี Node/npm แค่ดับเบิลคลิกไฟล์ `index.html` ก็ใช้ได้ทันที หรือจะโฮสต์บน GitHub Pages ก็ได้ในคลิกเดียว
- **📅 ปฏิทินภาษีอัจฉริยะ** — คำนวณวันครบกำหนดยื่นภาษีถัดไปให้อัตโนมัติ (ภ.พ.30, ภ.ง.ด.1/3/53, ภ.ง.ด.94, ภ.ง.ด.51, ภ.ง.ด.90/91, ภ.ง.ด.50) พร้อมนับถอยหลัง แจ้งเตือนผ่านเบราว์เซอร์ และ **export เป็นไฟล์ .ics** เพื่อซิงก์เข้า Google/Outlook Calendar ได้เลย
- **🕘 ระบบประวัติในตัว** — บันทึกผลการคำนวณแต่ละครั้งไว้เปรียบเทียบย้อนหลัง โหลดค่าที่เคยกรอกกลับมาคำนวณใหม่ได้ในคลิกเดียว หรือส่งออกเป็น CSV ไปวิเคราะห์ต่อ
- **🎯 Preset สถานการณ์จริง** — มนุษย์เงินเดือน, ฟรีแลนซ์, มีครอบครัว/บุตร กดปุ่มเดียวกรอกข้อมูลตัวอย่างให้ทันที ลองเล่นได้โดยไม่ต้องคิดเลขเอง
- **🌙 โหมดกลางคืนที่อ่านง่ายจริง** — ออกแบบสีให้ตัดกันชัดเจน ไม่ปวดตาแม้ใช้งานตอนดึก
- **🖨️ พิมพ์ใบสรุปสวยงามระดับมืออาชีพ** — ใส่ชื่อ-เลขประจำตัวผู้เสียภาษี พร้อมตราประทับและตารางขั้นบันไดภาษี พิมพ์ออกมาใช้วางแผนหรือแนบเอกสารได้ทันที
- **🎨 ประสบการณ์ใช้งานที่ไม่น่าเบื่อ** — แอนิเมชันตัวเลข, มาตรวัดความหนักภาษี (mood gauge), กราฟขั้นบันไดภาษีแบบ interactive และเอฟเฟกต์ฉลองเมื่อภาษีเป็นศูนย์ 🎉

## 🚀 เริ่มใช้งาน

```bash
git clone https://github.com/<your-username>/<your-repo>.git
cd <your-repo>
```

จากนั้นเปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ที่คุณใช้ประจำ — จบ ไม่มีขั้นตอนอื่นอีก

หรือจะเปิดใช้งานผ่านเว็บได้ทันทีด้วย **GitHub Pages**: ไปที่ `Settings → Pages` ของ repository แล้วตั้งค่า branch ที่ต้องการ deploy

## 🗂️ โครงสร้างโปรเจกต์

```
.
├── index.html   # โครงสร้างหน้าเว็บ
├── style.css    # ดีไซน์และธีม (รวมโหมดกลางวัน/กลางคืน)
└── script.js    # ตรรกะการคำนวณภาษี, ปฏิทิน, ประวัติ
```

ไฟล์แยกกันชัดเจน แก้ไข/ต่อยอดง่าย ไม่ต้องงมหาโค้ดในไฟล์เดียวยาวเป็นพันบรรทัด

## 🛠️ เทคโนโลยีที่ใช้

- HTML5 / CSS3 / Vanilla JavaScript (ไม่มี framework, ไม่มี build tool)
- Web Storage API (`localStorage`) สำหรับเก็บประวัติและโปรไฟล์ผู้เสียภาษีในเครื่อง
- iCalendar (.ics) format สำหรับ export ปฏิทินภาษี
- Google Fonts: Noto Serif Thai, Noto Sans Thai, IBM Plex Mono
